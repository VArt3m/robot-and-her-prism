# robot-and-her-prism

A 2-D light-beam puzzle. The **active, canonical implementation is `robot_prism_html/`** (browser port, pure ES modules — source in `robot_prism_html/js/`, tests at `robot_prism_html/`).

> **Work only in `robot_prism_html/`.** The sibling `robot_prism_py/` and the loose `robot_prism.py` are a **legacy, outdated** tkinter prototype — superseded by the HTML port and no longer maintained. **Do not read, reference, or port from them.** Treat `robot_prism_html/` as the single source of truth.

For all navigation, patterns, and run/test instructions, see @robot_prism_html/CLAUDE.md.
